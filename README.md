# Late-For-Work

A platformer where time constantly slows down, so you need to collect items to keep time going.

Play here: https://lionclaws.itch.io/late-for-work 

<img width="1706" height="902" alt="Appealing Image" src="https://github.com/user-attachments/assets/03e06a49-e2e8-4281-b5b8-6fc23b432f14" />


Collect items such as coffee and caffeine pills to speed up time before you fall asleep. You have three lives, so make the best of them or start the entire game all over again. 

There are two different difficulties you can choose from: Easy mode and Normal mode. Easy mode has time slow down less rapidly than normal mode, but the main platforming difficulty is the same. It is meant to be a fast paced game finished in around 5 minutes. 

Entire game made using GDScript in Godot with a total of 12 different scripts for each part of the 15 different scenes.
