# Late-For-Work

A platformer where time constantly slows down, so you need to collect items to keep time going.

Play here: https://lionclaws.itch.io/late-for-work

# About the game

Collect items such as coffee and caffeine pills to speed up time before you fall asleep. You have three lives, so make the best of them or start the entire game all over again. 

There are two different difficulties you can choose from: Easy mode and Normal mode. Easy mode has time slow down less rapidly than normal mode, but the main platforming difficulty is the same. It is meant to be a fast paced game finished in around 5 minutes. 

# Code Details

Entire game made using GDScript and GDShader in Godot with a total of 12 different scripts for each part of the 15 different scenes. 

Only 1 script was used with GDShader (being the Vignette/shader), making the majority of the code GDScript, however they are technically distinct.

Longest script is the "Player" script, with many bugs resulting from the movement rather than the central mechanic of the game. Coyote time and double jumps are very difficult to code together, making for some possible unintended bugs, however the major ones were patched within the first half of coding the project.
